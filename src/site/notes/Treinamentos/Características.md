## Apresentação das Características

### Nome (Custo)

### Requisito
Esta categoria lista qualquer Característica, Palavra-chave ou outra condição necessária para aplicar esta Característica a um Movimento Especial. Se uma Característica exigir uma determinada Palavra-chave e outra Característica remover a Palavra-chave necessária para esta Característica, esta Característica não poderá ser aplicada ao movimento. É importante atender aos requisitos de Características já adicionados ao adicionar outra Característica que altere uma Palavra-chave, a fim de evitar a criação de um movimento inválido. Por exemplo, um movimento já possui uma Característica Negativa que exige a Palavra-chave Ofensiva. Então, o jogador decide adicionar uma Característica Positiva com a Palavra-chave Utilitário. Isso remove implicitamente a Palavra-chave Ofensiva, tornando a Característica Negativa anterior inválida.
### Palavra-Chave
Esta categoria lista todas as Palavras-chave adicionadas ou removidas do Movimento Especial. Algumas Palavras-chave adicionadas podem remover implicitamente outras, como por exemplo, a Palavra-chave Sem Dano.
### Instâncias
Se a Característica puder ser escolhida mais de uma vez, esta categoria indica quantas vezes ele pode ser aplicado a um único Movimento Especial. Se esta categoria não estiver presente, a Característica só poderá ser escolhido uma vez por movimento.
### Combo
Esta seção inclui quaisquer regras especiais que se aplicam a um movimento com esta Característica quando usado em um Combo, bem como quaisquer Palavras-chave relacionadas a Combos.
### Descrição
A descrição da Característica, detalhando seus efeitos
